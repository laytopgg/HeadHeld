# HeadHeld

**HeadHeld makes player interactions clearer and more expressive by displaying held items above the character’s head — perfect for multiplayer visibility, PvP awareness, and roleplay immersion.**

## ✨ Features
- 🧍‍♀ Instantly shows items from both hands above the player
- 👀 Quickly understand what others are holding — no guessing
- 🎮 Fully client-side — works on any server
- ⚙️ Clean and simple configuration
- 🚀 Lightweight and performance-friendly

## 🎰 Why use HeadHeld?
Ever missed what another player was holding in a critical moment?

HeadHeld solves that instantly:
- Spot weapons, tools, or items at a glance
- Improve reaction time in PvP
- Enhance communication in RP and co-op
- Make multiplayer feel more alive and readable

## ⚙️ Customization
Tailor the display to your preference:
- Adjust position and height
- Change item scale
- Toggle display behavior

## 🧹 Compatibility
- ✔ Fabric
- ✔ Client-side only
- ✔ Works on any server (no installation required server-side)

## 📦 Dependencies
- Fabric API
- ModMenu (for in-game configuration)
