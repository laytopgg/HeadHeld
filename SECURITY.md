# Security Policy

## Supported Versions
Currently, only the latest version of HeadHeld is supported. Please ensure you are running the most recent release before reporting a security issue.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it directly through GitHub's [security advisories](https://github.com/laytopgg/HeadHeld/security/advisories). This allows us to address the issue before it becomes public. Do not disclose vulnerabilities publicly until a fix has been released.
