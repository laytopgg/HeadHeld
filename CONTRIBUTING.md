# Contributing to HeadHeld

Thanks for taking the time to contribute!

We welcome bug reports, feature requests, and pull requests from the community.

## How to contribute

1. **Search the issues**. Before opening a new issue, please check the [open issues](../../issues) to see if someone has already reported something similar. Feel free to add a comment if you have additional context.
2. **Open an issue**. If you don't see your bug or request already, [open a new issue](../../issues/new/choose) using the appropriate template.
3. **Fork the repository**. For code changes, fork the repository, create a new branch with a descriptive name, and commit your changes.
4. **Test your changes**. Make sure your changes work as expected and don't break existing functionality.
5. **Submit a pull request**. Open a pull request against the `main` branch with a clear description of your changes and the issue they address.

## Development environment

HeadHeld is built with Fabric for Minecraft. To set up a development environment:

1. Clone this repository.
2. Install the [Fabric API](https://modrinth.com/mod/fabric-api) and [Fabric Loader](https://fabricmc.net/).
3. Import the project into your IDE of choice (e.g., IntelliJ IDEA) and ensure dependencies resolve.
4. Run or debug using the provided Gradle tasks.

## Code of Conduct

This project adheres to the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
